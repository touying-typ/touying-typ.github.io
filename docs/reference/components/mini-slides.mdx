# mini-slides

<Signature>
  <code><SignatureName>mini-slides</SignatureName>(<SignatureParam>[self: none](#self), </SignatureParam><SignatureParam>[fill: rgb("000000")](#fill), </SignatureParam><SignatureParam>[alpha: 60%](#alpha), </SignatureParam><SignatureParam>[display-section: false](#display-section), </SignatureParam><SignatureParam>[display-subsection: true](#display-subsection), </SignatureParam><SignatureParam>[linebreaks: true](#linebreaks), </SignatureParam><SignatureParam>[short-heading: true](#short-heading), </SignatureParam><SignatureParam>[inline: false](#inline)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/components.typ#L428" /></code>
</Signature>

Section navigation component showing all sections and their per-slide progress as small filled/empty circle dots.

Typically placed in a theme's page header. Each section is labeled with a link, and each slide within the section is represented by a small dot (filled for the current slide, hollow for others). The active section uses the full `fill` color; inactive sections have `alpha` transparency applied.

<Parameters>

#### <ParamName>self</ParamName> : `none` <Default>`none`</Default> {#self}

<Param>

The self context, used to resolve short headings.

</Param>

#### <ParamName>fill</ParamName> : `color` <Default>`rgb("000000")`</Default> {#fill}

<Param>

The text and dot color. Default is `rgb("000000")`.

</Param>

#### <ParamName>alpha</ParamName> : `ratio` <Default>`60%`</Default> {#alpha}

<Param>

The transparency applied to inactive sections. Default is `60%`.

</Param>

#### <ParamName>display-section</ParamName> : `bool` <Default>`false`</Default> {#display-section}

<Param>

Whether to show per-slide dots for level-1 section headings. Default is `false`.

</Param>

#### <ParamName>display-subsection</ParamName> : `bool` <Default>`true`</Default> {#display-subsection}

<Param>

Whether to show per-slide dots for level-2 subsection headings. Default is `true`.

</Param>

#### <ParamName>linebreaks</ParamName> : `bool` <Default>`true`</Default> {#linebreaks}

<Param>

Whether to insert a line break after section/subsection labels. Default is `true`.

</Param>

#### <ParamName>short-heading</ParamName> : `bool` <Default>`true`</Default> {#short-heading}

<Param>

Whether to shorten heading labels using `utils.short-heading`. Default is `true`.

</Param>

#### <ParamName>inline</ParamName> : `bool` <Default>`false`</Default> {#inline}

<Param>

Whether to place dots on the same line as the section label instead of below it. Default is `false`.

</Param>

</Parameters>

**Returns:** `content`
