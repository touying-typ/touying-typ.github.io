# side-by-side

<Signature>
  <code><SignatureName>side-by-side</SignatureName>(<SignatureParam>[columns: auto](#columns), </SignatureParam><SignatureParam>[gutter: 1em](#gutter), </SignatureParam><SignatureParam>[..bodies](#bodies)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/components.typ#L24" /></code>
</Signature>

A simple wrapper around `grid` that creates a single-row grid. Used as the default `composer` for multi-body slides.

Example: `side-by-side[a][b][c]` will display `a`, `b`, and `c` side by side.

<Parameters>

#### <ParamName>columns</ParamName> : `auto` | `array` <Default>`auto`</Default> {#columns}

<Param>

The column widths. Default is `auto`, which creates equal-width columns matching the number of bodies.

</Param>

#### <ParamName>gutter</ParamName> : `length` <Default>`1em`</Default> {#gutter}

<Param>

The space between columns. Default is `1em`.

</Param>

#### <ParamName>..bodies</ParamName> : `content` {#bodies}

<Param>

The contents to display side by side.

</Param>

</Parameters>

**Returns:** `content`
