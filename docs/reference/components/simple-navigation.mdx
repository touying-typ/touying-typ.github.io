# simple-navigation

<Signature>
  <code><SignatureName>simple-navigation</SignatureName>(<SignatureParam>[self: none](#self), </SignatureParam><SignatureParam>[short-heading: true](#short-heading), </SignatureParam><SignatureParam>[primary: white](#primary), </SignatureParam><SignatureParam>[secondary: gray](#secondary), </SignatureParam><SignatureParam>[background: black](#background), </SignatureParam><SignatureParam>[logo: none](#logo)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/components.typ#L577" /></code>
</Signature>

A horizontal navigation bar showing all level-1 sections as clickable links.

The active section label is shown in `primary` color; all other sections use `secondary` color. An optional logo is placed at the right edge. Typically used as a page header in themes.

<Parameters>

#### <ParamName>self</ParamName> : `none` <Default>`none`</Default> {#self}

<Param>

The self context, used to resolve short headings.

</Param>

#### <ParamName>short-heading</ParamName> : `bool` <Default>`true`</Default> {#short-heading}

<Param>

Whether to shorten heading labels using `utils.short-heading`. Default is `true`.

</Param>

#### <ParamName>primary</ParamName> : `color` <Default>`white`</Default> {#primary}

<Param>

The text color of the currently active section. Default is `white`.

</Param>

#### <ParamName>secondary</ParamName> : `color` <Default>`gray`</Default> {#secondary}

<Param>

The text color of inactive sections. Default is `gray`.

</Param>

#### <ParamName>background</ParamName> : `color` <Default>`black`</Default> {#background}

<Param>

The background fill of the navigation bar. Default is `black`.

</Param>

#### <ParamName>logo</ParamName> : `content` | `none` <Default>`none`</Default> {#logo}

<Param>

Optional logo displayed at the right side of the bar. Default is `none`.

</Param>

</Parameters>

**Returns:** `content`
