# config-methods

<Signature>
  <code><SignatureName>config-methods</SignatureName>(<SignatureParam>[init](#init), </SignatureParam><SignatureParam>[cover](#cover), </SignatureParam><SignatureParam>[uncover](#uncover), </SignatureParam><SignatureParam>[only](#only), </SignatureParam><SignatureParam>[effect](#effect), </SignatureParam><SignatureParam>[alternatives-match](#alternatives-match), </SignatureParam><SignatureParam>[alternatives](#alternatives), </SignatureParam><SignatureParam>[alternatives-fn](#alternatives-fn), </SignatureParam><SignatureParam>[alternatives-cases](#alternatives-cases), </SignatureParam><SignatureParam>[item-by-item](#item-by-item), </SignatureParam><SignatureParam>[alert](#alert), </SignatureParam><SignatureParam>[show-only-notes](#show-only-notes), </SignatureParam><SignatureParam>[convert-label-to-short-heading](#convert-label-to-short-heading), </SignatureParam><SignatureParam>[..args](#args)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/configs.typ#L428" /></code>

The configuration of the methods.

<Parameters>

#### <ParamName>init</ParamName> : `function` {#init}

<Param>

The function to initialize the presentation. It should be `(self: none, body) => { .. }`.

</Param>

#### <ParamName>cover</ParamName> : `function` {#cover}

<Param>

The function to cover content. The default value is `utils.method-wrapper(hide)` function. You can configure it with `cover: utils.semi-transparent-cover` to use the semi-transparent cover.

</Param>

#### <ParamName>uncover</ParamName> : `function` {#uncover}

<Param>

The function to uncover content. The default value is `utils.uncover` function.

</Param>

#### <ParamName>only</ParamName> : `function` {#only}

<Param>

The function to show only the content. The default value is `utils.only` function.

</Param>

#### <ParamName>effect</ParamName> : `function` {#effect}

<Param>

The function to add effect to the content. The default value is `utils.effect`.

</Param>

#### <ParamName>alternatives-match</ParamName> : `function` {#alternatives-match}

<Param>

The function to match alternatives. The default value is `utils.alternatives-match` function.

</Param>

#### <ParamName>alternatives</ParamName> : `function` {#alternatives}

<Param>

The function to show alternatives. The default value is `utils.alternatives` function.

</Param>

#### <ParamName>alternatives-fn</ParamName> : `function` {#alternatives-fn}

<Param>

The function to show alternatives with a function. The default value is `utils.alternatives-fn` function.

</Param>

#### <ParamName>alternatives-cases</ParamName> : `function` {#alternatives-cases}

<Param>

The function to show alternatives with cases. The default value is `utils.alternatives-cases` function.

</Param>

#### <ParamName>item-by-item</ParamName> : `function` {#item-by-item}

<Param>

The function to show items one by one. The default value is `utils.item-by-item` function.

</Param>

#### <ParamName>alert</ParamName> : `function` {#alert}

<Param>

The function to alert the content. The default value is `utils.method-wrapper(text.with(weight: "bold"))` function.

</Param>

#### <ParamName>show-only-notes</ParamName> : `function` {#show-only-notes}

<Param>

The function used to render speaker notes, either as the primary content (`show-only-notes: true` mode) or on a second screen. It should accept `(self: none, width: 0pt, height: 0pt, cutout: false)`. When `cutout: true`, return a dictionary with `background`, `foreground`, and `cutout-height` keys.

</Param>

#### <ParamName>convert-label-to-short-heading</ParamName> : `function` {#convert-label-to-short-heading}

<Param>

The function to convert label to short heading. It is useful for the short heading for heading with label. It will be used in function with `short-heading`. The default value is `utils.titlecase(lbl.replace(regex("^[^:]*:"), "").replace("_", " ").replace("-", " "))`. It means that some headings with labels like `section:my-section` will be converted to `My Section`.

</Param>

</Parameters>

**Returns:** `dictionary`
