# touying-slides

<Signature>
  <code><SignatureName>touying-slides</SignatureName>(<SignatureParam>[..args](#args), </SignatureParam><SignatureParam>[body](#body)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/slides.typ#L25" /></code>

Touying slides function.

Example:

```typst
#show: touying-slides.with(
  config-page(paper: "presentation-" + aspect-ratio),
  config-common(
    slide-fn: slide,
  ),
  ..args,
)
```

<Parameters>

#### <ParamName>..args</ParamName> : `arguments` {#args}

<Param>

The configurations of the slides. For example, you can use `config-page(paper: "presentation-16-9")` to set the aspect ratio of the slides.

</Param>

#### <ParamName>body</ParamName> : `content` {#body}

<Param>

The contents of the slides.

</Param>

</Parameters>

**Returns:** `content`
