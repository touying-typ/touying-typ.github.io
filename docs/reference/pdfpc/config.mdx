# config

<Signature>
  <code><SignatureName>config</SignatureName>(<SignatureParam>[duration-minutes: none](#duration-minutes), </SignatureParam><SignatureParam>[start-time: none](#start-time), </SignatureParam><SignatureParam>[end-time: none](#end-time), </SignatureParam><SignatureParam>[last-minutes: none](#last-minutes), </SignatureParam><SignatureParam>[note-font-size: none](#note-font-size), </SignatureParam><SignatureParam>[disable-markdown: false](#disable-markdown), </SignatureParam><SignatureParam>[default-transition: none](#default-transition)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/pdfpc.typ#L124" /></code>
</Signature>

Configuration for the pdfpc export. You can export the pdfpc file by shell command `typst query --root . ./example.typ --field value --one "<pdfpc-file>" > ./example.pdfpc`.

Example:

```typ
#pdfpc.config(
  duration-minutes: 30,
  start-time: datetime(hour: 14, minute: 10, second: 0),
  end-time: datetime(hour: 14, minute: 40, second: 0),
  last-minutes: 5,
  note-font-size: 12,
  disable-markdown: false,
  default-transition: (
    type: "push",
    duration-seconds: 2,
    angle: ltr,
    alignment: "vertical",
    direction: "inward",
  ),
)
```

<Parameters>

#### <ParamName>duration-minutes</ParamName> : `int` <Default>`none`</Default> {#duration-minutes}

<Param>

The duration of the presentation in minutes.

</Param>

#### <ParamName>start-time</ParamName> : `datetime` <Default>`none`</Default> {#start-time}

<Param>

The start time of the presentation.

</Param>

#### <ParamName>end-time</ParamName> : `datetime` <Default>`none`</Default> {#end-time}

<Param>

The end time of the presentation.

</Param>

#### <ParamName>last-minutes</ParamName> : `int` <Default>`none`</Default> {#last-minutes}

<Param>

The number of minutes to show the last slide.

</Param>

#### <ParamName>note-font-size</ParamName> : `float` <Default>`none`</Default> {#note-font-size}

<Param>

The font size of the speaker notes.

</Param>

#### <ParamName>disable-markdown</ParamName> : `bool` <Default>`false`</Default> {#disable-markdown}

<Param>

A flag to disable markdown in the speaker notes.

</Param>

#### <ParamName>default-transition</ParamName> : `dictionary` | `none` <Default>`none`</Default> {#default-transition}

<Param>

The default slide transition. A dictionary with optional keys: `type` (str, e.g. `"push"`), `duration-seconds` (int), `angle` (direction, e.g. `ltr`), `alignment` (str, `"horizontal"` or `"vertical"`), `direction` (str, `"inward"` or `"outward"`). Default is `none`.

</Param>

</Parameters>

**Returns:** `content`
