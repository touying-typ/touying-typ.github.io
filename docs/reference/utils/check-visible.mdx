# check-visible

<Signature>
  <code><SignatureName>check-visible</SignatureName>(<SignatureParam>[idx](#idx), </SignatureParam><SignatureParam>[visible-subslides](#visible-subslides)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/utils.typ#L1105" /></code>
</Signature>

Check if a subslide index is visible given a visibility specification.

Example: `check-visible(3, "2-")` returns `true`

<Parameters>

#### <ParamName>idx</ParamName> : `int` {#idx}

<Param>

The current subslide index.

</Param>

#### <ParamName>visible-subslides</ParamName> : `int` | `array` | `str` {#visible-subslides}

<Param>

Specifies which subslides are visible. Supported formats: - A single integer, e.g. `3` — only subslide 3. - An array, e.g. `(1, 2, 4)` — equivalent to `"1, 2, 4"`. - A string with ranges, e.g. `"-2, 4, 6-8, 10-"` — subslides 1, 2, 4, 6, 7, 8, 10, and all after 10.

</Param>

</Parameters>

**Returns:** `bool`
