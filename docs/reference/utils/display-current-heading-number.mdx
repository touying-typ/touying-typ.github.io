# display-current-heading-number

<Signature>
  <code><SignatureName>display-current-heading-number</SignatureName>(<SignatureParam>[level: auto](#level), </SignatureParam><SignatureParam>[numbering: auto](#numbering), </SignatureParam><SignatureParam>[hierachical: true](#hierachical), </SignatureParam><SignatureParam>[depth: 9999](#depth)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/utils.typ#L533" /></code>
</Signature>

Display the current heading number on the page.

<Parameters>

#### <ParamName>level</ParamName> : `int` | `auto` <Default>`auto`</Default> {#level}

<Param>

The level of the heading. If `level` is `auto`, it will return the last heading on or before the current page. If `level` is a number, it will return the last heading on or before the current page with the same level.

</Param>

#### <ParamName>numbering</ParamName> : `str` | `auto` <Default>`auto`</Default> {#numbering}

<Param>

The numbering of the heading. If `auto`, uses the heading's own numbering. If a string, uses that as the numbering pattern.

</Param>

#### <ParamName>hierachical</ParamName> : `bool` <Default>`true`</Default> {#hierachical}

<Param>

Whether to return the heading hierarchically. If `true`, returns the last heading according to the hierarchical structure. If `false`, returns the last heading on or before the current page with the same level.

</Param>

#### <ParamName>depth</ParamName> : `int` <Default>`9999`</Default> {#depth}

<Param>

The maximum depth of the heading to search. Usually, it should be set as slide-level.

</Param>

</Parameters>

**Returns:** `content`
