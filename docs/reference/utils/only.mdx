# only

<Signature>
  <code><SignatureName>only</SignatureName>(<SignatureParam>[self: none](#self), </SignatureParam><SignatureParam>[visible-subslides](#visible-subslides), </SignatureParam><SignatureParam>[only-cont](#only-cont)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/utils.typ#L1268" /></code>
</Signature>

Display content in some subslides only. No space is reserved when hidden.

**Example:**

```typst
#show: simple-theme.with(aspect-ratio: "16-9")
= Slide
#only("2")[Only on subslide 2]
```

<Parameters>

#### <ParamName>visible-subslides</ParamName> : `int` | `array` | `str` {#visible-subslides}

<Param>

A single integer, an array of integers, or a string specifying the visible subslides. Supported formats: - A single integer, e.g. `3` — only subslide 3. - An array, e.g. `(1, 2, 4)` — equivalent to `"1, 2, 4"`. - A string with ranges, e.g. `"-2, 4, 6-8, 10-"` — subslides 1, 2, 4, 6, 7, 8, 10, and all after 10.

</Param>

#### <ParamName>only-cont</ParamName> : `content` {#only-cont}

<Param>

The content to display when visible.

</Param>

</Parameters>

**Returns:** `content`
