# add-dicts

<Signature>
  <code><SignatureName>add-dicts</SignatureName>(<SignatureParam>[dict-a](#dict-a), </SignatureParam><SignatureParam>[dict-b](#dict-b)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/utils.typ#L12" /></code>

Add a dictionary to another dictionary recursively.

Example: `add-dicts((a: (b: 1)), (a: (c: 2)))` returns `(a: (b: 1, c: 2))`

<Parameters>

#### <ParamName>dict-a</ParamName> : `dictionary` {#dict-a}

<Param>

The base dictionary.

</Param>

#### <ParamName>dict-b</ParamName> : `dictionary` {#dict-b}

<Param>

The dictionary to merge into `dict-a`.

</Param>

</Parameters>

**Returns:** `dictionary`
