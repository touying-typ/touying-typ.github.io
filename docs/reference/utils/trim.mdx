# trim

<Signature>
  <code><SignatureName>trim</SignatureName>(<SignatureParam>[arr](#arr), </SignatureParam><SignatureParam>[empty-contents: ([], [ ], parbreak(), linebreak())](#empty-contents)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/utils.typ#L112" /></code>
</Signature>

Remove leading and trailing empty elements from an array of content.

Example: `trim(([], [ ], parbreak(), linebreak(), [a], [ ], [b], [c], linebreak(), parbreak(), [ ], [ ]))` returns `([a], [ ], [b], [c])`

<Parameters>

#### <ParamName>arr</ParamName> : `array` {#arr}

<Param>

The array of content to trim.

</Param>

#### <ParamName>empty-contents</ParamName> : `array` <Default>`([], [ ], parbreak(), linebreak())`</Default> {#empty-contents}

<Param>

An array of content elements considered empty. Default is `([], [ ], parbreak(), linebreak())`.

</Param>

</Parameters>

**Returns:** `array`
