# alpha-changing-cover

<Signature>
  <code><SignatureName>alpha-changing-cover</SignatureName>(<SignatureParam>[self: none](#self), </SignatureParam><SignatureParam>[alpha: 25%](#alpha), </SignatureParam><SignatureParam>[fallback-hide: true](#fallback-hide), </SignatureParam><SignatureParam>[transparentize-table: false](#transparentize-table), </SignatureParam><SignatureParam>[it](#it)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/utils.typ#L1018" /></code>
</Signature>

Cover content with an alpha-changing mechanism.

Example: `config-methods(cover: utils.alpha-changing-cover.with(alpha: 25%))`

<Parameters>

#### <ParamName>alpha</ParamName> : `ratio` <Default>`25%`</Default> {#alpha}

<Param>

The opacity to apply to text colors when covered. Default is `25%`.

</Param>

#### <ParamName>fallback-hide</ParamName> : `bool` <Default>`true`</Default> {#fallback-hide}

<Param>

Whether to hide the content if it does not contain text. Default is `true`.

</Param>

#### <ParamName>transparentize-table</ParamName> : `bool` <Default>`false`</Default> {#transparentize-table}

<Param>

Whether to transparentize table content. Default is `false`.

</Param>

#### <ParamName>it</ParamName> : `content` {#it}

<Param>

The content to cover.

</Param>

</Parameters>

**Returns:** `content`
