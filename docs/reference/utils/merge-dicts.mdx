# merge-dicts

<Signature>
  <code><SignatureName>merge-dicts</SignatureName>(<SignatureParam>[init-dict](#init-dict), </SignatureParam><SignatureParam>[..dicts](#dicts)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/utils.typ#L38" /></code>
</Signature>

Merge some dictionaries recursively.

Example: `merge-dicts((a: (b: 1)), (a: (c: 2)))` returns `(a: (b: 1, c: 2))`

<Parameters>

#### <ParamName>init-dict</ParamName> : `dictionary` {#init-dict}

<Param>

The initial dictionary to start from.

</Param>

#### <ParamName>..dicts</ParamName> : `array` {#dicts}

<Param>

Additional dictionaries to merge in order.

</Param>

</Parameters>

**Returns:** `dictionary`
