# color-changing-cover

<Signature>
  <code><SignatureName>color-changing-cover</SignatureName>(<SignatureParam>[self: none](#self), </SignatureParam><SignatureParam>[color: gray](#color), </SignatureParam><SignatureParam>[fallback-hide: true](#fallback-hide), </SignatureParam><SignatureParam>[transparentize-table: false](#transparentize-table), </SignatureParam><SignatureParam>[it](#it)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/utils.typ#L989" /></code>
</Signature>

Cover content with a text-color-changing mechanism.

Example: `config-methods(cover: utils.color-changing-cover.with(color: gray))`

<Parameters>

#### <ParamName>color</ParamName> : `color` <Default>`gray`</Default> {#color}

<Param>

The color to apply to text when covered. Default is `gray`.

</Param>

#### <ParamName>fallback-hide</ParamName> : `bool` <Default>`true`</Default> {#fallback-hide}

<Param>

Whether to hide the content if it does not contain text. Default is `true`.

</Param>

#### <ParamName>transparentize-table</ParamName> : `bool` <Default>`false`</Default> {#transparentize-table}

<Param>

Whether to transparentize table content. Default is `false`.

</Param>

#### <ParamName>it</ParamName> : `content` {#it}

<Param>

The content to cover.

</Param>

</Parameters>

**Returns:** `content`
