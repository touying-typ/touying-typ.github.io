# semi-transparent-cover

<Signature>
  <code><SignatureName>semi-transparent-cover</SignatureName>(<SignatureParam>[self: none](#self), </SignatureParam><SignatureParam>[alpha: 85%](#alpha), </SignatureParam><SignatureParam>[body](#body)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/utils.typ#L939" /></code>
</Signature>

Cover content with a semi-transparent rectangle matching the page background color.

Example: `config-methods(cover: utils.semi-transparent-cover)`

<Parameters>

#### <ParamName>alpha</ParamName> : `ratio` <Default>`85%`</Default> {#alpha}

<Param>

The opacity of the covering rectangle (higher means more opaque/more hidden). Default is `85%`.

</Param>

#### <ParamName>body</ParamName> : `content` {#body}

<Param>

The content to cover.

</Param>

</Parameters>

**Returns:** `content`
