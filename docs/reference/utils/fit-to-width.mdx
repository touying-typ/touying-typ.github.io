# fit-to-width

<Signature>
  <code><SignatureName>fit-to-width</SignatureName>(<SignatureParam>[grow: true](#grow), </SignatureParam><SignatureParam>[shrink: true](#shrink), </SignatureParam><SignatureParam>[width](#width), </SignatureParam><SignatureParam>[content](#content)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/utils.typ#L829" /></code>
</Signature>

Fit content to specified width.

Example: `#utils.fit-to-width(100%)[BIG]`

<Parameters>

#### <ParamName>grow</ParamName> : `bool` <Default>`true`</Default> {#grow}

<Param>

Indicates whether the content should be scaled up if it is smaller than the available width. Default is `true`.

</Param>

#### <ParamName>shrink</ParamName> : `bool` <Default>`true`</Default> {#shrink}

<Param>

Indicates whether the content should be scaled down if it is larger than the available width. Default is `true`.

</Param>

#### <ParamName>width</ParamName> : `length` | `fraction` | `relative` {#width}

<Param>

The width to fit the content to.

</Param>

#### <ParamName>body</ParamName> : `content` {#body}

<Param>

The content to fit.

</Param>

</Parameters>

**Returns:** `content`
