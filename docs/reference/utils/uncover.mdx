# uncover

<Signature>
  <code><SignatureName>uncover</SignatureName>(<SignatureParam>[self: none](#self), </SignatureParam><SignatureParam>[visible-subslides](#visible-subslides), </SignatureParam><SignatureParam>[uncover-cont](#uncover-cont), </SignatureParam><SignatureParam>[cover-fn: auto](#cover-fn)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/utils.typ#L1228" /></code>
</Signature>

Uncover content in some subslides. Reserved space when hidden (like `#hide()`).

**Example:**

```typst
#show: simple-theme.with(aspect-ratio: "16-9")
= Slide
#uncover("2-")[Only visible from subslide 2]
```

<Parameters>

#### <ParamName>visible-subslides</ParamName> : `int` | `array` | `str` {#visible-subslides}

<Param>

A single integer, an array of integers, or a string specifying the visible subslides. Supported formats: - A single integer, e.g. `3` — only subslide 3. - An array, e.g. `(1, 2, 4)` — equivalent to `"1, 2, 4"`. - A string with ranges, e.g. `"-2, 4, 6-8, 10-"` — subslides 1, 2, 4, 6, 7, 8, 10, and all after 10.

</Param>

#### <ParamName>uncover-cont</ParamName> : `content` {#uncover-cont}

<Param>

The content to display when visible.

</Param>

#### <ParamName>cover-fn</ParamName> : `function` | `auto` <Default>`auto`</Default> {#cover-fn}

<Param>

An optional cover function to use instead of the default cover method from the theme. Useful when using `uncover` inside external package integrations (e.g. `fletcher.hide` for fletcher diagrams).

</Param>

</Parameters>

**Returns:** `content`
