# touying-progress

<Signature>
  <code><SignatureName>touying-progress</SignatureName>(<SignatureParam>[callback](#callback)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/utils.typ#L81" /></code>
</Signature>

Get the progress of the current slide.

`utils.last-slide-number` gives the total slide count and can be used directly in headers or footers.

**Example:**

```typst
#show: simple-theme.with(aspect-ratio: "16-9")
= Slide
#touying-progress(ratio => {
"Progress: " + str(int(ratio * 100)) + "%"
})
```

<Parameters>

#### <ParamName>callback</ParamName> : `function` {#callback}

<Param>

A function `ratio => { .. }` receiving a float between `0.0` and `1.0`.

</Param>

</Parameters>

**Returns:** `content`
