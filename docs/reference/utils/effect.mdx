# effect

<Signature>
  <code><SignatureName>effect</SignatureName>(<SignatureParam>[self: none](#self), </SignatureParam><SignatureParam>[fn](#fn), </SignatureParam><SignatureParam>[visible-subslides](#visible-subslides), </SignatureParam><SignatureParam>[cont](#cont), </SignatureParam><SignatureParam>[is-method: false](#is-method)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/utils.typ#L1186" /></code>
</Signature>

Take effect in some subslides.

Example: `#effect(text.with(fill: red), "2-")[Something]` will display `[Something]` if the current slide is 2 or later.

You can also add an abbreviation by using `#let effect-red = effect.with(text.with(fill: red))` for your own effects.

<Parameters>

#### <ParamName>fn</ParamName> : `function` {#fn}

<Param>

The function that will be called in the subslide. Or you can use a method function like `(self: none) => { .. }`.

</Param>

#### <ParamName>visible-subslides</ParamName> : `int` | `array` | `str` {#visible-subslides}

<Param>

A single integer, an array of integers, or a string specifying the visible subslides. Supported formats: - A single integer, e.g. `3` — only subslide 3. - An array, e.g. `(1, 2, 4)` — equivalent to `"1, 2, 4"`. - A string with ranges, e.g. `"-2, 4, 6-8, 10-"` — subslides 1, 2, 4, 6, 7, 8, 10, and all after 10.

</Param>

#### <ParamName>cont</ParamName> : `content` {#cont}

<Param>

The content to display when the content is visible in the subslide.

</Param>

#### <ParamName>is-method</ParamName> : `bool` <Default>`false`</Default> {#is-method}

<Param>

Whether the function is a method function. Default is `false`.

</Param>

</Parameters>

**Returns:** `content`
