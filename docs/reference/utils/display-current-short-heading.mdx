# display-current-short-heading

<Signature>
  <code><SignatureName>display-current-short-heading</SignatureName>(<SignatureParam>[self: none](#self), </SignatureParam><SignatureParam>[level: auto](#level), </SignatureParam><SignatureParam>[hierachical: true](#hierachical), </SignatureParam><SignatureParam>[depth: 9999](#depth), </SignatureParam><SignatureParam>[setting: body => body](#setting), </SignatureParam><SignatureParam>[style](#style), </SignatureParam><SignatureParam>[..setting-args](#setting-args)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/utils.typ#L575" /></code>
</Signature>

Display the current short heading on the page.

<Parameters>

#### <ParamName>level</ParamName> : `int` | `auto` <Default>`auto`</Default> {#level}

<Param>

The level of the heading. If `level` is `auto`, it will return the last heading on or before the current page. If `level` is a number, it will return the last heading on or before the current page with the same level.

</Param>

#### <ParamName>hierachical</ParamName> : `bool` <Default>`true`</Default> {#hierachical}

<Param>

Whether to return the heading hierarchically. If `true`, returns the last heading according to the hierarchical structure. If `false`, returns the last heading on or before the current page with the same level.

</Param>

#### <ParamName>depth</ParamName> : `int` <Default>`9999`</Default> {#depth}

<Param>

The maximum depth of the heading to search. Usually, it should be set as slide-level.

</Param>

#### <ParamName>style</ParamName> : `function` {#style}

<Param>

The style of the heading. If `style` is a function, it will use the function to style the heading. For example, `style: (self: none, current-heading) => utils.short-heading(self: self, current-heading)`.

</Param>

</Parameters>

**Returns:** `content`
