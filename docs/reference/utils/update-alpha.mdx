# update-alpha

<Signature>
  <code><SignatureName>update-alpha</SignatureName>(<SignatureParam>[color](#color), </SignatureParam><SignatureParam>[alpha](#alpha)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/utils.typ#L925" /></code>
</Signature>

Update the alpha channel of a color.

Example: `update-alpha(rgb("#ff0000"), 0.5)` returns a red color with 50% opacity.

<Parameters>

#### <ParamName>color</ParamName> : `color` {#color}

<Param>

The color to update.

</Param>

#### <ParamName>alpha</ParamName> : `ratio` {#alpha}

<Param>

The new alpha value as a percentage (e.g. `50%` for half-transparent).

</Param>

</Parameters>

**Returns:** `color`
