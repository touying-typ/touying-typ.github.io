# bibliography-as-footnote

<Signature>
  <code><SignatureName>bibliography-as-footnote</SignatureName>(<SignatureParam>[numbering: "[1]"](#numbering), </SignatureParam><SignatureParam>[bibliography](#bibliography), </SignatureParam><SignatureParam>[body](#body)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/magic.typ#L164" /></code>
</Signature>

Display bibliography citations as footnotes. Place `#place(hide(bibliography(...)))` at the end of the document to register the bibliography entries.

Usage: `#show: magic.bibliography-as-footnote.with(bibliography(title: none, "ref.bib"))`

<Parameters>

#### <ParamName>numbering</ParamName> : `str` <Default>`"[1]"`</Default> {#numbering}

<Param>

The numbering format for footnote citations. Default is `"[1]"`.

</Param>

#### <ParamName>bibliography</ParamName> : `bibliography` {#bibliography}

<Param>

The bibliography element, e.g. `bibliography("ref.bib")`.

</Param>

</Parameters>

**Returns:** `content`
