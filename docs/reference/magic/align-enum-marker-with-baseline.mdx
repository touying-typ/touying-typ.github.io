# align-enum-marker-with-baseline

<Signature>
  <code><SignatureName>align-enum-marker-with-baseline</SignatureName>(<SignatureParam>[body](#body)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/magic.typ#L46" /></code>
</Signature>

Apply as a show rule to vertically align enum markers with the baseline of the first line of each enum item. Only works for numeric markers (e.g. `1.`).

Usage: `#show: align-enum-marker-with-baseline`

**Returns:** `content`
