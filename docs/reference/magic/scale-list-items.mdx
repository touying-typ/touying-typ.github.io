# scale-list-items

<Signature>
  <code><SignatureName>scale-list-items</SignatureName>(<SignatureParam>[scale: .75](#scale), </SignatureParam><SignatureParam>[body](#body)</SignatureParam>)<SourceLink href="https://github.com/touying-typ/touying/blob/main/src/magic.typ#L94" /></code>
</Signature>

Scale the font size of nested list, enum, and terms items.

Usage: `#show: scale-list-items.with(scale: .75)`

<Parameters>

#### <ParamName>scale</ParamName> : `int` | `float` <Default>`.75`</Default> {#scale}

<Param>

The font size ratio of the current nesting level relative to the parent. Default is `.75`.

</Param>

#### <ParamName>body</ParamName> : `content` {#body}

<Param>

The content to apply the scaling to.

</Param>

</Parameters>

**Returns:** `content`
